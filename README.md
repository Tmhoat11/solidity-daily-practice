# Solidity Daily Practice  
本仓库用于记录我每日的 Solidity 练习。  
目标：建立从零到“可交付合约”能力。  

## 使用说明  
- 每天创建一个新的文件夹 `dayN-xxx`，N 表示天数。  
- 文件夹内包含：`xxx.sol`（合约代码） + `README.md`（今日学习摘要）  
- 每次练习内容要包含：  
  * 今日理解的关键点  
  * 今日遇到的疑问 & 解决方式  
  * 明日 ACTION：下一步将做什么  

## 学习成果展示  
- 写合约 → 部署 → 测试 → 报错 → 总结  
- 逐步添加至主项目仓库 `solidity-projects`
